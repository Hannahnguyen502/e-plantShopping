# Paradise Nursery - Online Plant Shopping Application

Paradise Nursery is a dynamic, user-friendly e-commerce web application built to provide plant enthusiasts with a seamless shopping experience. Users can browse a diverse collection of indoor and outdoor plants, view detailed descriptions, and effortlessly manage a real-time shopping cart.

This project was developed as a final capstone to demonstrate proficiency in building interactive, stateful front-end applications using modern web development practices.

---

## 🚀 Features

### 1. Landing Page
* An elegant, welcoming introduction to Paradise Nursery.
* A clear call-to-action button ("Get Started") that seamlessly transitions users to the main product page.

### 2. Product Listing Page
* **Categorized Plants:** Items are intuitively grouped (e.g., Air Purifying, Low Maintenance, Aromatic) to enhance the browsing experience.
* **Detailed Product Cards:** Displays high-quality plant images, names, descriptions, and prices.
* **Instant Interaction:** An "Add to Cart" button that dynamically updates the global navigation cart counter upon clicking and disables itself once selected to prevent accidental duplicates.

### 3. Interactive Shopping Cart
* **Dynamic Counter:** The navigation bar keeps a real-time count of the total number of items in the cart.
* **Cart Management:** Users can increase or decrease item quantities directly inside the cart or delete items entirely.
* **Automatic Cost Calculation:** Subtotals for individual plant types and the grand total for the entire order update instantly as changes are made.
* **Seamless Navigation:** A "Continue Shopping" button allows users to quickly jump back to the plant listing without losing their cart state.
* **Checkout Simulation:** A responsive "Checkout" button that alerts the user that checkout functionality is coming soon.
